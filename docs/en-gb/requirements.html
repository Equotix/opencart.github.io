System / Server Requirements

System Requirements

OpenCart has certain technical requirements to be met for the store to operate properly. A web server must be setup to make the OpenCart store publicly available on the web. Domain names and hosting services can easily be purchased at an affordable price.

For testing purposes, you can setup the store locally on your PC as well.

When choosing a web hosting provider, you should check to ensure that the server provided meets the requirements below.

  • Web Server (Apache / NGINX)
  • PHP 7.3+
  • Database (MySQLi / MariaDB)

Required PHP libraries / modules

  • cURL
  • ZIP
  • Zlib
  • GD Library
  • Mbstring
  • Xml

The above PHP extensions should be readily available on most servers by hosting providers. During the installation process, OpenCart will check whether you have them all enabled. You should contact your hosting provider if any is missing.
