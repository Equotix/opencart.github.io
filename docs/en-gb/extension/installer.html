Extension Installer

The OpenCart extension installer enables you to install any additional extensions that you may need by uploading the zip package.

Installing an Extension

Extensions can be downloaded from the OpenCart marketplace in a zip package. If it is compatible with the installer, you will just need to click on the upload button at the top right of the page, and the installer will do the rest of the work for you.

The OpenCart Extension Installer.

Uninstalling an Extension

You can simply click on the "Uninstall" button if you wish to uninstall any extensions that were installed using the Extension Installer. By clicking on the uninstall button, all files that were added will be automatically deleted.

Deleting an Extension

Once the extension has been uninstalled, you can click on the "Delete" button to delete the extension from the installer.
