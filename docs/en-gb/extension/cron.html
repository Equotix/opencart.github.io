Cron

A cron is a service that allows the system to automatically execute a script at a specific time without human intervention. This page will list all the cron tasks added by extensions in your store.

Setting up Cron Jobs

You will need to add a cron task to run hourly on your server to trigger OpenCart's cron system. Depending on the control panel you use, the steps varies. The cron command to be added is shown at Extensions > Cron Jobs.

The cron job command to be setup.

Using Plesk

Using Command Line

Add the below line to your crontab file. Replace the "{cron command here}" text with the command copied from your OpenCart store.

0 * * * * {cron command here}
