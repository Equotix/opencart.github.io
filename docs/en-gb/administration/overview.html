Admin Interface

OpenCart's administration panel is where you can modify features, upload images, add products, keep track of customers, manage payments, and much more. Customisation in the admin affects how your customer will interact with your store. You can do so by modifying the look, structure, and content of the store front.

Accessing the Admin Panel

To access the admin panel, type in your store' URL into the web browser followed by "/admin". For example, if your store is located at "www.chocolatechip.com", your admin panel is located at "www.chocolatechip.com/admin". Even if the store is located in a sub-folder or on a sub-domain of their site, adding "/admin" to the end of the store's path will lead you to the administration panel.

If you've changed the name of the folder after the installation, the location of the admin panel is changed to that new name. For example, if the name of the admin folder is changed to "mysecretadmin", the new location of the admin panel is at "www.chocolatchip.com/mysecretadmin".

Changing the name of the admin folder to another name is important to improve your online store's security. See Basic Security Practices for more information.

A username and password was created during Step 3 of the installation. This information can now be used to fill in the administrator login details.

OpenCart store admin login interface.

After filling in the correct username and password, pressing the "Login" button will direct you to your OpenCart dashboard. When you first login to your stop, the dashboard will be blank, because there isn't any statistical data to be shown yet.

Dashboard Modules

The dashboard is the first thing you will see when you login as an admin. The main function of the dashboard is to give the store owner an overview of how the store is performing. The dashboard modules can be rearranged and configured under the Dashboard Extensions as well. Below are the initial configured dashboard modules.

  • Total Orders

    This displays the total orders for the store.

  • Total Sales

    This displays the total sales for the store.

  • Total Customers

    This displays the total customers for the store.

  • People Online

    This displays the current number of visitors on your website.

  • World Map

    The world map displays the geographic location for all your orders.

  • Sales Analytics

    A graph is provided to track the chronological progress of the store relative to the amount of orders and customers over time. The X axis is time, which can be hours, days, or months depending on the range selected. The Y axis displays the number of total orders and total customers.

  • Recent Activity

    The recent activities from yours customers on your store such as login, creating account or placing new orders is displayed here.

  • Latest Orders

    This displays the recent orders and some of their order information.

Admin Menu

The administration navigation is located at the side of your admin panel. You can navigate between the various configurations and pages of the admin using the menu. These configurations will be explained in the subsequent sections of the User Guide.
