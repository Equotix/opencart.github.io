Customers

Shop owners should know who their customers are and how to manage their information. In the administration, customer information will need to be stored efficiently to remember any transactions made with their account. To access customer information, you can log into the administration panel of the OpenCart store. The Customer management sections are located under "s;Customers" in the main menu.

Customers Listing Page

When a customer creates an account or completes the checkout process (opting to sign up as your customer), OpenCart will store their information automatically. A complete list of customers will be displayed on this page. You can also add a new customer by clicking on the "Add" button at the top right of the page.

The page listing all your customers.

Creating a Customer

Customer accounts usually don't need to be created in the administration panel, since account information is stored automatically upon customer sign up at the store front. However, should you need to manually add customers, you can click on the "Add" button at the top right of the page. You will then be presented with a form with the following fields.

General Tab
Store This is the store the customer created his account on.
Customer Group This is the customer group this customer belongs to.
First Name * This is the customer's first name.
Last Name * This is the customer's last name.
E-mail This is the customer's email address.
Telephone This is the customer's telephone number.
Password This is the customer's account login password. If editing, leave this blank to not change it.
Confirm Re-enter the password above if you are creating a new password, else leave it blank.
Newsletter This is the customer's newsletter subscription status.
Status This is the customer's account status. Disabled and the customer will not be able to login.
Safe Set this to "Yes" to prevent the customer from being flagged by the anti-fraud system if you have enabled it.
Addresses
First Name * The first name for this address.
Last Name * The last name for this address.
Company The company for this address.
Address 1 * The address line 1 for this address.
Address 2 The address line 2 for this address.
City * The city for this address.
Postcode * The postcode for this address. It can be optional, depending on your configuration for the country in System > Localisation > Countries.
Country The country for this address.
Region / State The region / state for this address.
Default Address Set this to make it the default selected address when the customer checkout.
* Required field

The form for adding or editing a customer.

History Tab

The history tab allows you to add notes to a customer's account. This allows you to leave history for future references you need on the customer.

Transactions Tab

The transactions tab allows you to add or subtract store credits from the customer's account. The store credits can be used to offset an order during checkout. The transactions can be seen in the customer's account at the store front too.

Reward Points

The reward points tab allows you to add or subtract reward points from the customer's account. The reward points can be used to offset an order during checkout if a product is configured to support the reward points. Additionally, you will need to enable the reward points feature in OpenCart too.

IP Addresses Tab

An IP Address will reveal the location of customer's computer network. There are many website services available to shop owners, that will give the location for an IP address. The list of IP addresses the customer has used on your website is logged and displayed in this tab.
