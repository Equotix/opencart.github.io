Customer Groups

Customer Groups provide a greater organisational scheme for the customer list. This is helpful in selecting customer groups for discounted products and special offers in the store, and for sending out emails to specific groups of customers.

Adding a Customer Group

When creating a new customer group, you will be presented with the fields below.

Customer Group Name * This is the name of the customer group.
Description This is a description of the group for internal use.
Approve New Customers Setting this to "Yes" means that all customers will need to be manually approved by the admin before they can login if they selected this customer group during sign up.
Sort Order This defines the arrangement of this customer group.
* Required field
