Custom Fields

Custom fields allows you to add additional fields to the customer sign up form, address form, or the affiliate form. These fields will collect additional information that you may require from the customers or the affiliates.

Creating a Custom Field

Get started by clicking the "Add" button at the top right of the page to add a new field. The following fields will then be shown.

Custom Field Name * This is the name of the custom field which will be displayed to the customers.
Location This is the location where the field will be displayed.
Type This is the type of input field which will be used. The various types of fields available will be explained below.
Customer Group This custom field will be displayed for the selected customer groups.
Required This custom field will be required for the selected customer groups.
Status This is the status of the custom field. If disabled, it will not be shown.
Sort Order This is the sort order for the custom field. A smaller number will have it be displayed higher up in the list.
Custom Field Values This is only applicable for select, radio and checkbox type fields.
* Required field

The form for creating a custom field.

Field Types

The various field types are explained below.

Select

Selecting this will display a select dropdown option on the form, allowing only one value to be selected at a time.

An example of a select type field.

Radio

Selecting this will display radio buttons on the form, similar to the select type which only allows one value to be selected at a time.

An example of a radio type field.

Checkbox

Selecting this will display checkboxes on the form, allowing multiple values to be selected.

An example of a checkbox type field.

Text

Selecting this will display a single row input box for customers to type any text.

An example of a text type field.

Textarea

Selecting this will display a multi rows input box for customers to type any amount of text.

An example of a textarea type field.

File

Selecting this will display a file upload button.

An example of a file type field.

Date

Selecting this will display a date input box with a date picker.

An example of a date type field.

Time

Selecting this will display a time input box with a time picker.

An example of a time type field.

Date & Time

Selecting this will display a date and time input box with a date-time picker.

An example of a date & time type field.
