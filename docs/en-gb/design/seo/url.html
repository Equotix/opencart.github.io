SEO URL

The SEO URL list shows all the SEO keywords you have created in your store. This allows you to easily find and make any changes required to the page URL.

Creating a SEO URL

You can create a new SEO URL for any page on your website. The fields are explained below.

Store * This is the store this URL will apply to on a multi-store setup.
Language This is the language this URL will apply to.
Key * This is the GET parameter key that this URL will match.
Value * This is the GET parameter value that this URL will match based on the above key.
Keyword This keyword value will be used to set the URL if the above key and value matches.
* Required field
