SEO Profiles

SEO Profiles are used only when your SEO URL mode is enabled in System > Settings. It is used to rewrite your website's URL to a search engine friendly format. The SEO Profiles created by OpenCart during installation is sufficient for most general purpose use, and third party plugins should add into it whenever necessary. If you need to append additional GET parameter keys for specific URLs, you can add or modify the SEO Profiles here.

Creating a SEO Profile

Elaborating on how the SEO Profile works, we will make use of the default installed "Product" profile. Below is an explanation of each field.

Profile Name * This is the name of the profile for internal reference only.
Key * This is the GET parameter key that OpenCart will look for in the URL.
Regex Regular expression is used to match the key set above.
Push If there is a match on the key based on the regular expression above, this information is pushed into the GET parameter of the URL.
Remove If there is a match on the key based on the regular expression above, this information is removed from the GET parameter of the URL. Comma separated keys are supported.
Sort Order The order in which the profiles should run to handle the URL.
* Required field

The form for adding a new SEO Profile.

Based on the above "Product" profile that comes with the default OpenCart installation, it can be understood that this profile is used to handle the SEO URL for the product page.

It first checks if the key "product_id" is available in the URL, and then uses regular expression to match for digits in the key value. Once it matches, it will push the route product/product back into the URL, and remove the route and product_id parameters. This allows the URL to be clean and link correctly to the SEO keyword you've set for the correct product.
