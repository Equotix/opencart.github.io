Language Editor

Language editor allows you to easily make changes to the text being displayed at your store front. Simply add a new translation to override the default value provided.

Creating a Translation

Add a translation by clicking on the "add new" button. You will then be presented with a set of fields, which will be explained below.

Store This is the store that the translation will be used on for a multi-store setup.
Language This will display a list of languages that's installed.
Route This is a list of language files that's available on your website which you can override.
Key The translation texts that are available in the file is shown once the route above has been selected.
Default The default text that is in the language file is shown here.
Value Enter a new text that you wish to use to override the above default.
