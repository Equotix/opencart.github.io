Banners

Banners created are a collection of pictures containing linkable images displayed on the store's store front. Banners can be utilized to catch the customer's eye and direct them to special products or promotions in the store. The Banners here allows you customise the images and links displayed on the banners.

Creating a Banner

Certain modules, including third party modules, may use the banners created here. Simply create theme or modify them according to your needs.

Banner Name * This is the name of the collection which you will use to identify it in the modules.
Status This sets the banner status to make it available for use.
Title * This is the title of the image.
Link This is the link for this image which you want it to redirect to when clicked.
Image This is the image to be shown.
Sort Order This defines the arrangement of this image within this banner group.
* Required field
