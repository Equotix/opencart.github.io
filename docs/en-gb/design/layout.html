Layouts

Layouts is used for attaching modules to the specific page you require. OpenCart comes pre-installed with a set of layouts. If you require more layouts, you can create them here as well, then attach it to specific products, categories, information pages or manufacturers. This will give you the ability to configure a different layout for the specific page you attach the layout to.

Creating a Layout

You can create a new layout simply by clicking on the "add" button at the top of the page. You will then be presented with a set of fields, which will be explained below.

Layout Name * This is the name of the layout, which you can use to identify it in the design tab (available in products, categories, information pages and manufacturers) which gives you the ability to set the layout override.
Store This is for multi-store support. Select the store this layout should be used on.
Route Enter the route this layout will be used on, or leave it empty if you intend to use it as a layout override The route can be identified by following the instructions below.
Modules Position Assign modules to the various positions on the layout accordingly to design your website. You need to click on the "add module" button to add it the preferred position.
* Required field

The form for adding a layout.

Identifying the Route

The "route" field may be confusing to some. Here's how you can identify the route to use.

  1. Load your website's store front.
  2. Navigate to the page you want to use the layout on.
  3. Look at your website's address bar on the browser.
  4. The "route=" attribute will identify the route you need to use for the page.
