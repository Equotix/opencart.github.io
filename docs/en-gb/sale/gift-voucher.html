Gift Vouchers

Gift Vouchers works similarly to a gift card. Your customers can purchase gift vouchers and send it to friends, or you can create gift vouchers and share the voucher code for customers to use. The vouchers has stored values and works like your store credits, where customers can spend partially and keep it for future orders.

There is currently no expiry date for a gift voucher in OpenCart.

Creating a Voucher

You will be presented with the following fields to create a voucher.

Code * This is the code to be applied during checkout.
From Name * This is the name of the sender for the gift voucher.
From Email * This is the title of the image.
To Name * This is the name of the receiver for the voucher.
To Email * This is the email address of the receiver.
Theme The gift voucher theme. You can create or edit the available voucher themes in Sales &glt; Gift Vouchers &glt; Voucher Themes.
Message This is the gift voucher message that will be sent with the email.
Amount This is the stored value amount for the gift voucher.
Status This is the status for the gift voucher. If disabled, the code cannot be used. If enabled, it needs to have a balance for the voucher to be usable too.
* Required field

The form for adding or editing a voucher.

Voucher Themes

The Voucher Themes section alters the image displayed in the email according to the theme. At the store front, customers purchasing the voucher can choose a theme for the voucher.

The voucher theme page for adding or editing the available voucher themes.

Voucher Email

Once a gift voucher has been created, you are responsible for sending out the email to the customer receiving the gift voucher. After you make sure the customer's email information has been filled up correctly in the voucher, you can send the email by clicking "Send". Once the email has been sent, OpenCart will display the message: "Success: Gift Voucher e-mail has been sent!".
