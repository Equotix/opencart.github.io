Returns

The return page allows you to manage your order returns. The order returns can be requested by the customer directly from their account dashboard or the admin can manually add it on this page too.

Adding / Editing a Return

Adding or editing a return will show you the same form with the fields below.

Order Information
Order ID * This is the order ID that is being returned.
Order Date This is the order date.
Customer This is the customer requesting for the return. If set, this will show up in the customer's account too.
First Name * The first name for the order return.
Last Name * The last name for the order return.
Email * The email address for the order return.
Telephone * The telephone number for the order return.
Product Information & Reason for Return
Product * The product being returned.
Model * The model number for the product being returned.
Quantity The number of quantity being returned.
Return Reason The reason requested for returning the order. The reasons can be changed in System > Localisation > Returns > Return Reasons.
Opened The status of the product, whether it has been opened.
Comment Additional comments for the return request.
Return Action The action taken for the return request. The actions can be changed in System > Localisation > Returns > Return Actions.
Return Status The status for the return request. The status can be changed in System > Localisation > Returns > Return Statuses.
* Required field

Return details

Filling in the Returns Details tab is the first step in creating a new return. The following details are required to fill out a return:

  • Order ID: the ID of the order that was returned.
  • Order Date: the date that order was made.
  • Customer
  • First Name
  • Last Name
  • E-mail
  • Telephone
