Recurring Orders

Recurring orders are payments that your customers are billed on a recurring basis, dependent on the recurring schedule you've setup.

You will first need to setup a Recurring Profile. Once setup and a customer makes an order with a recurring profile attached to it, the order information will show up on this page.

The Recurring Orders page will list the orders with the following information.

Recurring ID A number OpenCart automatically generates.
Order ID The order ID this recurring order is associated to.
Payment Reference The ID of the recurring profile in PayPal Express (by default).
Customer The name of the customer subscribed to this recurring order in your store.
Status The status of the payment in PayPal.
Date Added The date this recurring order was added.

The list of recurring orders.

Searching Recurring Orders

The "Filter" panel on the right helps you find the recurring orders based on the information specified in the search fields. It can help you locate recurring orders by date added, status, associate order ID or even the customer. For more information on the Filter tool, see Filter.
