Attributes Overview

Attributes are being used by products to provide customers with a comparison. It is always a good practice to have universal attributes and they can be created here. The attributes created here can then be assigned to the relevant products. Attributes can be used to describe the size, weight, colour or any other specifications of the product that have potential for comparison.

The product comparison page when two products with attributes are compared.

Setting Up Attributes

Getting started with attributes, you will first need to setup the attribute groups before you create the attributes. Head over to the Attribute Groups page to start creating your first attribute group.
