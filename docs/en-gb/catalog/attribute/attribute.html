Attributes

Once the attribute groups have been created, attributes can be created here and added to the relevant groups.

Attributes Listing Page

The available attributes are listed on this page. You can either delete or add more attributes on this page.

The attributes listing page.

Creating an Attribute

You can simply click on the "Add New" button at the top right of the listing page to get started. You will then be presented with a couple of fields, which is further explained below.

Attribute Name * This is the name of the attribute which will be shown.
Attribute Group This is the group this attribute will be assigned to.
Sort Order This allows you to specify the position of the filter value. The value "1" would take precedence as compared to the values "2" or "3".

The attributes form page.

Deleting an Attribute

Attributes cannot be deleted until it has been unassigned from all products. As such, if you wish to delete an attribute, you will first need to remove it from all linked products.
