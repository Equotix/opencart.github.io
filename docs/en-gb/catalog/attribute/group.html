Attribute Groups

Attribute Groups organise the different attributes into groups for easy maintenance. By placing attributes into the appropriate groups, you will find it easier when you are trying to modify products. For example, attributes such as "Clockspeed" and "No. of Cores" can be organised under the larger attribute group of "Processor".

Attribute Groups Listing Page

The available attribute groups are listed on this page. You can either delete or add more attribute groups on this page.

The attribute groups listing page.

Creating an Attribute Group

You can simply click on the "Add New" button at the top right of the listing page to get started. You will then be presented with two basic fields, which is further explained below.

Attribute Group Name * This is the name of the attribute group which will be shown.
Sort Order This allows you to specify the position of the filter value. The value "1" would take precedence as compared to the values "2" or "3".

The attribute groups form page.

Deleting an Attribute Group

Attribute groups cannot be deleted until all attributes have been unassigned from it. As such, if you wish to delete an attribute group, you will first need to remove any attributes that are linked to it.
