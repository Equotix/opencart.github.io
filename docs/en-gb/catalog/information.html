Information

Information is used to provide your customer with details that you think they might need. Information can be pages such as terms and conditions of your store, store address map or any other information that you think may be useful for the customer. Information pages can be displayed at the footer of your website too.

Information Listing Page

On the information listing page, you will have the option to edit, delete, or insert a new information page.

About Us, Delivery Information, Privacy Policy, and Terms & Conditions comes with the default installation because they are the most important information pages used in an online store. We recommend that you edit these pages rather than delete them.

The page displaying the information pages.

Create an Information Page

You can put just about anything into an information page. The available fields are explained below.

General Tab

The fields available in the general tab are explained below.

Information Title * This is the title of the page.
Description * This is the information page text.
Meta Tag Title * This is the title of the page which the information page is associated with. This is used by search engine results pages such as Google to display preview snippets of your page.
Meta Tag Description This would be a short description used by search engines, and will be seen as a short description under their search results.
Meta Tag Keywords These are keywords used by search engines to rank your page according to relevancy when a customer searches for certain keywords.
* Required field

The information General tab.

Data Tab

The fields available in the data tab are explained below.

Stores This is used to specify which store this information page should be linked to (this only applies when you are managing more than one store with OpenCart's multi-store feature).
Bottom This is used to specify whether the information page should appear in the footer of the store front.
Status This is used to set the status for this information page. Disabling will remove it from the store front.
Sort Order This allows you to specify the position of the information page displayed on the menu. The value "1" would take precedence as compared to the values "2" or "3".

The information page General tab.

SEO Tab

This tab is the same as the products and categories SEO tab. It asks for the following information.

Keyword Assign a unique keyword to make your information page URL search engine friendly. It is recommended that you use all lowercase characters, and replace spaces with a hyphen.

The information SEO tab.

Design Tab

The Information design tab is also the same as as the categories design tab. It allows you to configure the "Layout Override" feature. In this feature, you can choose to override a layout, such as using an "Account" page layout for this specific Information page instead of the default "Information" layout. For more information on layouts, see Modules. You can just leave it blank if you do not wish to override the layout.

The information design tab.
