Product Option Tab

Options are extra selections that a customer can make on the product page before adding the product to the shopping cart. This could be a size or colour of the product. Options are first created under Catalog > Options before being assigned to the product.

Simply type the name of the option you wish to add to this product, and the autocomplete field will appear for you to make a selection. Once selected, the option will be displayed for you to add the option values. Only the select, radio and checkbox type options can have an option value added to it.

The product option tab.

Once you click on the "Add Option Value" button, a popup with multiple fields will appear for you to fill it up. The fields are explained below. Click on the "Save" button when you are done to close the popup.

Option Value This is the selection value for the customer to choose from on the product page.
Quantity This is the available quantity for this option value that you have for customers to purchase.
Subtract Stock This is similar to the subtract stock field that is available in the "Data" tab. If set to "Yes", OpenCart will automatically deduct stocks from this option value whenever an order is made.
Price This is the price adjustment field when this option value is selected. Either add or subtract an amount from the product price.
Points This is the points adjustment field when this option value is selected. Either add or subtract the points from the product points configured in the rewards tab.
Weight This is the weight adjustment field when this option value is selected. Either add or subtract the weight from the product weight.
