Products Links

The links tab specifies which pages this product will be shown on or is related to. The fields are explained below.

Manufacturer This is used to indicate the product's manufacturer and link the product to it. A manufacturer has to be created beforehand if linking is required. Linking this product to the manufacturer will show it on the store front's manufacturer page.
Categories This is used to link the categories to the product. The product will be displayed under the selected categories. A category has to be created beforehand if linking is required.
Filters This is used to link the created filters to this product.
Stores This is used to specify which store this product should be linked to. This only applies if you are managing more than one store using the multi-store feature built into OpenCart.
Downloads This is used only when there is a downloadable product for purchase. You will need to create the download file beforehand.
Related Products This is used to identify similar products in your web store. The selected products will be displayed on this product's page under the related section at the store front.
