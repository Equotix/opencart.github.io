Product Special Tab

The Special tab is similar to the Discount tab, except that this is a special offer price without a need for bulk quantity purchase. The various fields are explained below.

Customer Group This is used to specify which customer group is entitled to the discount. Guest customers without a customer group will be assigned to the default customer group which is set in System > Settings.
Priority This is used to specify which discount would apply first if there are multiple discounts for the selected item. The priority number "1" will take precedence over the priority number "2" or "3"
Price This is the discounted price to be used.
Date Start This is used to specify when the discounted price will become effective. You can leave it empty, which means it starts right away.
Date End This is used to specify when the discounted price will cease. You can leave it empty, which means there is no end date.
