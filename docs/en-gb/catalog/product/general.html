Product General

The format of the General tab is identical to the General tab when creating a category, with the addition of the Product Tags field. The various fields to be filled up is explained below.

Product Name * This is the name of the product which would be displayed. If you are using OpenCart's built in multi-language feature, you will have to ensure that the different languages are filled up.
Description This is the text that will describe this product on the product page, which will be reflected on the store front when your customers visits the product page.
Meta Tag Title * This is the title of the page which the product is associated with. This is used by search engine results pages such as Google to display preview snippets of your page.
Meta Tag Description This would be a short description used by search engines, and will be seen as a short description under their search results.
Meta Tag Keywords These are keywords used by search engines to rank your page according to relevancy when a customer searches for certain keywords.
Product Tags These should be the relevant keywords that your customer searches for. If it matches the defined tags, then the product would appear in the search results. Do take note that you need to use a comma to separate each keyword (e.g. keyword 1, keyword 2, keyword 3).
* Required field
