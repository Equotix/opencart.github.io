Product Data

The data tab includes more technical data about the product. The fields are further explained below.

Model
Model * This allows you to create a model code for your product using combinations of letters and numbers. This model number need not be unique across all your products.
SKU Stock Keeping Unit. A random code for your products.
UPC Universal Product Code.
EAN European Article Number.
JAN Japanese Article Number.
ISBN International Standard Book Number.
MPN Manufacturer Part Number.
Location This is to specify where the product is located at.
Price
Price This is used to indicate the amount which you are selling the product for in the default store currency.
Tax Class This is used to indicate if any taxes should apply to this product. Taxes can be managed under the store's Tax page.
Stock
Quantity This is used to specify the amount of stock which is available for this product.
Minimum Quantity This is used to indicate the minimum quantity that a customer has to purchase for each checkout.
Subtract Stock This is to indicate whether the quantity should be deducted each time a customer makes an order.
Out of Stock Status This is used to specify a message to display on the product page when the quantity reaches 0 or less. You can configure the available stock statuses under the store's Stock Statuses page.
Date Available This is used to specify the starting date which the product will be made available for your customers to purchase on the store.
Specification
Requires Shipping This is used to specify if shipping is required for the product. This will allow you to collect the delivery address and enable the shipping methods on the checkout page if it is set to "Yes".
Dimensions This is used to specify the size of the product, which may be used by certain shipping modules to calculate the shipping cost.
Length Class This is used to indicate the metric units used for the measurements above.
Weight This is used to specify how heavy the product is, which may be used by certain shipping modules to calculate the shipping cost.
Weight Class This is used to indicate the metric units used for the weight above.
Status This allows you to specify if the product is enabled or disabled for purchase on the store front.
Sort Order This allows you to specify the position of the product displayed on the store front. The value "1" would take precedence as compared to the values "2" or "3".
Leaving "Sort Order" blank will organise the products in alphabetical order.
