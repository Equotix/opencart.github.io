Manufacturers

Manufacturers, also known as "Brands", allows you to group the products from the same brand together. This would allow your customers to easily identify products according to their favourite brands.

Manufacturers Listing Page

The manufacturers listed below came with the default installation. They can be deleted and new manufacturers can be added in its place. The manufacturers should be created before you add a product, similar to the categories. Click on the "Add New" button to get started with creating a new manufacturer.

The manufacturers listing page.

Creating a Manufacturer

Creating a manufacturer is pretty much similar to creating a category. Get started by filling up the manufacturer's information.

General Tab

The fields available in the general tab are explained below.

Manufacturer Name * This is the name of the manufacturer which will be shown.
Stores This is used to specify which store this manufacturer should be linked to (this only applies when you are managing more than one store with OpenCart's multi-store feature).
Image This is used to set the image or logo for this manufacturer.
Sort Order This allows you to specify the position of the manufacturer displayed on the menu. The value "1" would take precedence as compared to the values "2" or "3".

The manufacturer General tab.

SEO Tab

This tab is the same as the products and categories SEO tab. It asks for the following information.

Keyword Assign a unique keyword to make your manufacturer page URL search engine friendly. It is recommended that you use all lowercase characters, and replace spaces with a hyphen.

The manufacturer SEO tab.

Design Tab

The manufacturer design tab is also the same as as the categories design tab. It allows you to configure the "Layout Override" feature. In this feature, you can choose to override a layout, such as using an "Account" page layout for this specific manufacturer page instead of the default "Manufacturer" layout. For more information on layouts, see Modules. You can just leave it blank if you do not wish to override the layout.

The manufacturer design tab.

Assigning to a Product

You will also need to assign the manufacturer to the relevant product. To do that, you have to access Catalog > Products > Links tab and add the manufacturer to the product if you have yet to do so.

Store Front

The manufacturers feature has its own dedicated page at the store front. It can be accessed from the link at the footer, which is shown as "Brands". Clicking on the link will bring the customer to your brands directory, and will display all the available brands that you have in the store.

The dedicated manufacturers directory at the store front.

Clicking on any of the links in the directory will bring the customer to the dedicated manufacturer page which will display all the available products that has been linked to this manufacturer.
