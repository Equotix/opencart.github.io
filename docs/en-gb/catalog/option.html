Options

Options are extra selections that you can provide customers with under each of the product. This allows the customer to customise the product before adding it to their shopping cart. For example, if your product has multiple colours, you can add a "Colour" option for the customer to choose the colour he prefers before adding to the cart.

Options Listing Page

You can view all the available options in the Options page under the Catalog menu. Once you are ready to create a new option, you can click on the "Add New" button at the top right of the page.

The page displaying the available options in your store.

Creating an Option

There are many types of options that OpenCart supports. This can be configured here to suit what you require for the product. The fields available are explained below.

Option
Option Name * This is the name of the option which will be displayed.
Type This is the type of the option that will be displayed at the store front. The various types available will be covered below.
Sort Order This allows you to specify the position of the option. The value "1" would take precedence as compared to the values "2" or "3".
Option Values (Only applicable to "Choose" type options)
Option Value Name * This is the name of the option value that will be displayed.
Image This is the image that will be displayed beside the selection if the type of option supports it.
Sort Order This allows you to specify the position of the option value. The value "1" would take precedence as compared to the values "2" or "3".
* Required field

Option Types

The various option types are explained below.

Select

Selecting this will display a select dropdown option on the product, allowing only one value to be selected at a time.

An example of a select type option.

Radio

Selecting this will display radio buttons on the product, similar to the select type which only allows one value to be selected at a time.

An example of a radio type option.

Checkbox

Selecting this will display checkboxes on the product, allowing multiple option values to be selected.

An example of a checkbox type option.

Text

Selecting this will display a single row input box for customers to type any text.

An example of a text type option.

Textarea

Selecting this will display a multi rows input box for customers to type any amount of text.

An example of a textarea type option.

File

Selecting this will display a file upload button.

An example of a file type option.

Date

Selecting this will display a date input box with a date picker.

An example of a date type option.

Time

Selecting this will display a time input box with a time picker.

An example of a time type option.

Date & Time

Selecting this will display a date and time input box with a date-time picker.

An example of a date & time type option.
