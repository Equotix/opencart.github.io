Filters

Filters allow customers to filter the products in a more organised way when they are viewing the products at the store front. It will list the filters which you have created for them to select from.

Filters Listing Page

You can view all the available filters in the Filters page under the Catalog menu. Once you are ready to create a new filter group, you can click on the "Add New" button at the top right of the page.

The page displaying the available filters.

Creating a Filter

You will be presented with a page for you to create your filter group and the available values that belongs to the group on this page. For example, if a customer is looking for clothes on your store that is red in colour, he can easily filter for products with the filter tag "Red" assigned to it when you have the filter setup properly. The available fields are explained below.

Filter Group
Filter Group Name * This is the name of the filter group.
Sort Order This allows you to specify the position of the filter group. The value "1" would take precedence as compared to the values "2" or "3".
Filter Values
Filter Name * This is the filter value name.
Sort Order This allows you to specify the position of the filter value. The value "1" would take precedence as compared to the values "2" or "3".
* Required field

The page for setting up a filter group.

Assigning to a Category

Once the filter group has been created, you need to assign the values available for use to the respective category at Catalog > Categories > Data tab for it to be visible on the category page.

Assigning to a Product

You will also need to assign the filter values to the relevant product. To do that, you have to access Catalog > Products > Links tab and add the filter values to the product.

Setting Up the Filter Module

For the filters to be displayed, you will need to install the Filter module and assign it to a layout.
