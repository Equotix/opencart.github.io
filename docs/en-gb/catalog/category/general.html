Category General

The General tab asks for basic information regarding the product category. These fields are explained below.

Category Name * This is the name of the category which would be displayed. If you are using OpenCart's built in multi-language feature, you will have to ensure that the different languages are filled up.
Description This is the text that will describe this category on the category page, to be viewed by the customers in the store front of the shop.
Meta Tag Title * This is the title of the page which the category is associated with. This is used by search engine results pages such as Google to display preview snippets of your page.
Meta Tag Description This would be a short description used by search engines, and will be seen as a short description under their search results.
Meta Tag Keywords These are keywords used by search engines to rank your page according to relevancy when a customer searches for certain keywords.
* Required field
