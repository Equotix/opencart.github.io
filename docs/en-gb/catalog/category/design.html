Category Design Tab

The design tab consists of the "Layout Override" feature. In this feature, you can choose to override a layout, such as using an "Account" page layout for this specific category instead of the default "Category" layout. For more information on layouts, see Modules. You can just leave it blank if you do not wish to override the layout.
