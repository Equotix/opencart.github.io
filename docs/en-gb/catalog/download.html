Downloads

If you have downloadable files to attach to your products, it should be created here. Only then, your customers will be able to retrieve these downloadable files when they purchase a product linked with the files.

Customers will be required to create an account during checkout and will not be able to checkout as a guest if the product he is purchasing has a download linked to it.

Downloads Listing Page

The list of available downloads that you have created will be displayed on this page. You can get started by clicking on the "Add New" button at the top right of the page.

The list of available downloads.

Creating a Download

Adding a download file is straightforward with just a couple of clicks. The fields available are explained below.

Download Name * This is the name of the download.
Filename This is the actual filename of the file stored on the server. You can enter the filename if you uploaded the file through FTP, or just click on the upload button beside this field to select a file to upload.
Mask This is the filename that will be shown when the customer downloads the file.
* Required field

The form for adding a download.

Assigning to a Product

You will also need to assign the newly created download file to the relevant product. To do that, you have to access Catalog > Products > Links tab and add the download file to the product.

Downloading at Store Front

When a customer makes an order with a download file attached to the product, he will be able to download the file from the "Downloads" page of his account.

The downloadable file will not be shown on the product page. Hence, it is a good idea to describe how the customer will receive his file in the product description. For example, you can write a message in the product description such as "Your file can be downloaded from your account Downloads page once you've made the purchase."
